# Chat Application – Primary Reference

## Overview

This project is a real-time chat application built as part of an internship. It follows a full-stack architecture using modern web technologies to enable seamless one-to-one communication, user management, and secure authentication workflows.

The system is designed with a clear separation between client and server, supporting scalability and maintainability.

---

## Technology Stack

* **Frontend:** React.js
* **Backend:** Express.js (Node.js)
* **Real-time Communication:** Socket.IO
* **Database:** MongoDB

## Core Features

* **Real-time One-to-One Chat**
  Enables instant communication using WebSocket-based architecture via Socket.IO.

* **User Profile Management**
  Supports CRUD operations for user profiles, including updates to personal information.

* **Email OTP Verification**
  Implements user verification using Google SMTP to ensure authenticity during registration.

---

## Scope & Intent

This project demonstrates:

* Implementation of real-time systems using event-driven architecture
* Integration of authentication and verification mechanisms
* Structured full-stack development with modular separation
* Practical application of system design concepts (UI, DB, and architecture alignment)
